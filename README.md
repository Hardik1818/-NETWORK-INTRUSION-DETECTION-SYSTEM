## 🚀 Step-by-Step Setup

### Step 1 — Train the Models (Google Colab)

1. Open `UNSW_NB15_IDS_Classification.ipynb` in [Google Colab](https://colab.research.google.com)
2. Set Runtime → **T4 GPU**
3. Run all cells — the notebook downloads the dataset, trains RF + SVM + MLP, and saves all models
4. At the end, **Cell 18** downloads `UNSW_NB15_IDS_Results.zip`
5. Extract the zip — it contains the `saved_models/` folder

### Step 2 — Set Up This Repo

```bash
git clone https://github.com/<your-username>/<your-repo>
cd <your-repo>

# Copy saved_models from Colab download
cp -r /path/to/saved_models ./saved_models

# Install dependencies
pip install -r requirements.txt
```

### Step 3 — Run Locally

```bash
streamlit run app/streamlit_app.py
```

Open [http://localhost:8501](http://localhost:8501)

## 🌐 Deploy to Streamlit Community Cloud (Free)

1. Push this repo to GitHub (including `saved_models/`)
2. Go to [share.streamlit.io](https://share.streamlit.io) and sign in with GitHub
3. Click **New app**
4. Select your repo, branch `main`, and set:
   - **Main file path:** `app/streamlit_app.py`
5. Click **Deploy!**

Your app will be live at `https://<your-app-name>.streamlit.app` in ~2 minutes.

> ⚠️ **Note:** Streamlit Community Cloud has a 1 GB memory limit.  
> The MLP + Random Forest together may approach this. If you hit memory issues,  
> comment out the SVM model load in `streamlit_app.py`.
