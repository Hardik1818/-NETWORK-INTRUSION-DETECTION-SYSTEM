2. Set Runtime в†’ **T4 GPU**

## рџЊђ Deploy to Streamlit Community Cloud (Free)

1. Push this repo to GitHub (including `saved_models/`)
2. Go to [share.streamlit.io](https://share.streamlit.io) and sign in with GitHub
3. Click **New app**
4. Select your repo, branch `main`, and set:
   - **Main file path:** `app/streamlit_app.py`
5. Click **Deploy!**

Your app will be live at `https://<your-app-name>.streamlit.app` in ~2 minutes.

> вљ пёЏ **Note:** Streamlit Community Cloud has a 1 GB memory limit.  
> The MLP + Random Forest together may approach this. If you hit memory issues,  
> comment out the SVM model load in `streamlit_app.py`.
